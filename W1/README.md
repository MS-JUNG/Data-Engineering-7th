#### GDP ETL Pipeline Overview

이 프로젝트는 Wikipedia의 국가별 명목 GDP 데이터를 수집하여,
국가–대륙(Region) 정보를 결합한 뒤 정제된 JSON 파일로 저장하는 ETL 파이프라인입니다.

main()
 ├─ load_country_region_map()
 │   └─ country_region_map.json 로드
 │
 ├─ Extract
 │   └─ extract_gdp_table_html()
 │       ├─ fetch_html()
 │       │   └─ Wikipedia HTML 요청
 │       ├─ find_gdp_table_html()
 │       │   └─ GDP 테이블(table HTML) 탐색
 │       └─ _pick_latest_imf_column()
 │           └─ 최신 IMF 연도 컬럼 결정
 │
 ├─ Transform
 │   └─ transform_gdp_table()
 │       ├─ pd.read_html() 로 테이블 로드
 │       ├─ 국가명 정규화 (normalize_country_name)
 │       ├─ GDP 값 정제 및 단위 변환
 │       ├─ 국가 → Region 매핑
 │       ├─ 결측/매핑 실패 데이터 제거
 │       └─ 정렬 및 imf_year 컬럼 추가
 │
 └─ Load
     └─ save_json()
         └─ Countries_by_GDP.json 저장



# W1-M1
#### Q. 이런 데이터셋을 분석해서 얻을 수 있는 경제적 가치는 무엇일까요? 어떤 비즈니스 상황에서 이런 데이터셋을 분석해서 어떤 경제적 가치를 얻을 수 있을까요? 변수들 간의 상관 관계가 높은 조합을 임의로 2개 선택해서 해당 데이터 간의 상관 관계를 그래프로 그리고 어떤 결론을 내릴 수 있는지를 토의하세요.

같은 차량 데이터도 다음과 같이 위치에 따라 분석 방향을 달리한다.
- 제조사 관점: 성능·연비·원가 간 트레이드오프를 분석해 최적의 차량 설계 포인트를 도출
- 판매자 관점: 구매자 니즈(퍼포먼스형·연비형 등)에 부합하는 라인업 중 수익성을 극대화할 수 있는 차량 우선순위 제시
- 컨설팅 관점: 과거와 현재 데이터를 비교 분석해 기업의 제품 전략 및 포트폴리오 방향성 진단

그 중 제조사 관점에서의 경제적 가치 창출 시나리오를 진행해보았다.

#### 시나리오
자동차 제조사는 신규 차량 모델을 기획하는 과정에서 항상 비슷한 질문을 반복해서 마주한다.
“연비를 조금 더 올리면 성능은 얼마나 포기해야 할까?” “차량 무게를 줄이면 체감 성능에도 영향이 있을까?”
설계 회의에서는 경험에 기반한 의견이 오가지만, 서로 다른 설계 방향을 두고 명확한 근거를 제시하기는 쉽지 않다.
특히 연비와 성능처럼 서로 영향을 주는 요소들은 직관만으로 판단하기 어려운 경우가 많다.
이러한 상황에서 당신은 자동차 제조사에서 근무하는 데이터 분석가로서, 과거에 출시되었던 차량들의 성능과 제원이 정리된 데이터를 활용해
설계 변수 간의 관계를 분석해 달라는 요청을 받게 된다.


#### 분석 목표
분석의 목적은 단순히 수치를 비교하는 것이 아니라,
다음과 같은 질문에 데이터로 답하는 것이다.
차량 무게나 마력이 연비에 실제로 어떤 영향을 미치는가?
성능을 높이기 위해 감수해야 하는 연비 손실은 어느 정도인가?
비슷한 차량들 사이에서 설계 차이를 만드는 핵심 변수는 무엇인가?
이를 위해 연비(mpg), 마력(hp), 무게(wt), 실린더 수(cyl) 등의 변수를 포함한
차량 성능 데이터(mtcars)를 분석하고, 변수 간 상관관계를 통해 설계 요소 간의 특징을 정리한다.

비슷한 연비(mpg)의 차량들 중 hp가 높게 나타나는 차량들의 특징(실린더 수, 엔진 구조 등)을 분석해 연비 제약 조건 하에서 마력 성능을 최대화하는 설계 요소를 파악한다. 그리고 이를 차량 설계에 반영해 상품성이 좋은 신차를 제작한다. 동일 연비 내에서 높은 마력을 가진 차량 Duster 360, Ford Pantera L, Maserati Bora이 비슷한 연비를 가진 차량들에 비해 무게가 가벼우며 수동 변속기를 가짐을 분석을 통해 확인하였다.

#### 시사점 
과거 차량 데이터 분석을 통해 동일한 연비 조건에서도 무게 절감과 변속기 구성 최적화로 마력 성능을 향상시킬 수 있음을 확인하였다. 이러한 분석은 성능–연비 트레이드오프를 정량화할 뿐만 아니라, 향후 다양한 지표와 최신 데이터로 확장 가능한 설계 전략을 수립할 수 있다.
